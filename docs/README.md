# 미션 - 자동차 경주

# 기능 목록

---

## Model

### Name 
- isValid => 이름 조건 검증

### Move
- generateRandom => 랜덤값 생성

### Result
- calculateDistance => 이동거리 계산

---

## View

### InputView
- name => 이름 입력
- attemptsNum => 게임 횟수 입력

### OutputView
- showResult => 자동차 이름 + 이동 거리 출력

---

## Controller

### GameController
- start => 게임 시작
- playGame => 게임 실행 반복
- showResult => 결과 도출
